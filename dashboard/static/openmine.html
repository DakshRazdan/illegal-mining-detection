Detection Mathematics — Formulas, Weights & Statistics METHODOLOGY
SPECTRAL INDICES
NDVI = (NIR − Red) / (NIR + Red)
     = (B08 − B04) / (B08 + B04)
BSI = (SWIR + Red − NIR − Blue)
      / (SWIR + Red + NIR + Blue)
NDWI = (Green − NIR) / (Green + NIR)
      = (B03 − B08) / (B03 + B08)
NBR = (NIR − SWIR) / (NIR + SWIR)
     = (B08 − B11) / (B08 + B11)
Range: [−1, +1] · Normalised before fusion
WEIGHTED FUSION SCORE
S = Σ wi · norm(ΔIndexi)
   where ΔIndex = After − Before
wNDVI  = 0.35   vegetation loss
wBSI   = 0.30   bare soil gain
wNDWI = 0.20   water turbidity
wNBR   = 0.15   land disturbance
Σ weights = 1.00 ✓
Threshold τ = 0.50 · Flag if S > τ
